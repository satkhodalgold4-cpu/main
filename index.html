About the App

Satkhodal Gold App is a private internal application developed for securely uploading files to Google Drive using Google OAuth 2.0 APIs. This app does not provide public login functionality and is used only by the app owner for internal operations.

Application Usage

The application uses a refresh token to generate access tokens for uploading files to Google Drive. No Google Sign-In is used and no personal user data is collected, stored, or shared.

The app requests the minimum required Google Drive scope strictly for file upload functionality.
